## Quick Start

1. **Send intro email** — Copy email template from disclosure guide
2. **Schedule consultation** — 30-min call with Discovery to Impact
3. **Submit disclosures** — Via [Tradespace Portal](https://app.tradespace.io/signin/inventor?account_slug=d74c2a7194a7b03da184)
4. **Await approval** — Then update LICENSE files
